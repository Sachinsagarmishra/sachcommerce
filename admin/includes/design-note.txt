ADMIN PANEL DESIGN SYSTEM (DEALPORT UI)
---------------------------------------

FONTS:
- Montserrat: Use for Headings (h1-h6), Sidebar Brand, and table headers.
- Poppins: Use for Body text, Nav links, and UI elements.

COLOR PALETTE:
- Primary Brand: #4EA674 (Ocean Green) - Used for primary buttons, active states, and brand logo.
- Secondary Brand: #023337 (Cyprus) - Used for sidebar labels, main text, and secondary UI elements.
- Accent Background: #EAF8E7 (Aqua Spring) - Used for hover states and light backgrounds.
- Accent Hairline: #C1E6BA (Surf Crest) - Used for subtle borders and dividers.
- Secondary Primary: #6467F2 (Blue) - Used for alternative CTAs or links.

COMPONENT RULES:
1. Border Radius: Use 12px for buttons/inputs, 20px for cards/tables.
2. Icons: Use FontAwesome 6 (far/fas) with size 18px-20px.
3. Tables: Headers should have Aqua Spring background and Ocean Green text (all caps).
4. Sidebar: Sections should be labeled (Main menu, Product, Admin).
5. Cards: Use subtle box shadows: 0 4px 6px -1px rgba(0, 0, 0, 0.05).

MAINTENANCE:
- All global styles are defined in /admin/assets/css/admin.css.
- Use CSS Variables (:root) for color modifications.
- Ensure the .wrapper > .sidebar + .main-content > .admin-navbar + .content-wrapper structure is followed on all pages.
